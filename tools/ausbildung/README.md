# 中德技工桥接计划

这是一个面向中德双方客户的双语项目展示站点，用来验证德国技工需求与中国成熟技工出海之间是否存在可落地的项目机会。

## 当前版本

- 静态站点：`index.html`
- 双语切换：中文 / Deutsch
- 项目视觉资产：`assets/workforce-bridge.png`
- 主题：德国技工需求、短训桥接、项目制试点、季度产业回顾

## 执行材料

- `docs/90-day-roadmap.md`：90 天执行路线图。
- `docs/german-employer-interview.md`：德国企业访谈提纲。
- `docs/skilled-worker-profile-template.md`：中国技工能力档案模板。
- `docs/quarterly-industry-review-template.md`：季度产业回顾模板。

## 核心定位

这个项目不把短训包装成 Ausbildung 的替代品，也不把短训描述成签证捷径。它的作用是：

- 验证德国企业是否存在明确的项目型技工需求。
- 把中国技工经验整理成德国客户能理解的能力证据。
- 用 2-8 周短训补齐工艺标准、安全、质量记录、现场流程等差异。
- 通过小规模试点项目验证双方是否值得继续投入。

## 建议下一步

1. 按行业建立德国需求清单：建筑改造、暖通、电气、金属加工、机电、物流/现场服务。
2. 访谈 5-10 家德国中小企业，确认是否接受项目制、短期试点或国际技工能力验证。
3. 设计中国技工能力档案模板：证书、项目经历、设备经验、工艺视频、安全记录。
4. 请移民法、劳动法或行业协会顾问确认每类岗位的可行路径。
5. 每季度更新一次产业回顾，形成长期客户信任材料。
