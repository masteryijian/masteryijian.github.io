# 90 天执行路线图 / 90-Tage-Umsetzungsplan

## 项目目标 / Ziel

在 90 天内验证一个核心问题：德国企业是否愿意为来自中国的成熟技工能力购买短期、项目制、可监督的服务或试点岗位，而不是只接受长期 Ausbildung 路径。

Innerhalb von 90 Tagen soll geprüft werden, ob deutsche Unternehmen bereit sind, chinesische Praxiskompetenz projektbezogen, kurzfristig und kontrolliert einzusetzen, statt ausschließlich auf langfristige Ausbildungswege zu setzen.

## 第 1-2 周：需求假设 / Woche 1-2: Bedarfshypothesen

- 选定 3 个优先行业：建筑改造/暖通电气、金属/机电、物流/现场服务。
- 按联邦州、岗位、项目周期整理 20-30 个潜在需求场景。
- 建立德国企业候选名单：中小承包商、安装公司、能源改造服务商、工业维护服务商。
- 明确不碰或暂缓的高风险岗位：强监管职业、高安全责任岗位、语言不可替代岗位。

## 第 3-4 周：德国企业访谈 / Woche 3-4: Arbeitgeberinterviews

- 完成 8-12 个德国企业访谈。
- 每次访谈确认：是否缺人、缺什么能力、是否接受试点、可接受项目周期、合规顾虑、是否需要德语。
- 每次访谈都要识别背后购买动机：交付延期、成本压力、客户流失、旺季产能、Meister 负担、长期雇佣风险或行政负担。
- 访谈后立即给企业打标签：真实项目压力、付费意愿、试点开放度、合规复杂度、语言依赖度、内部推动人。
- 把需求分成三类：
  - A 类：可以进入试点设计。
  - B 类：有需求但合规/语言/资质门槛较高。
  - C 类：当前不适合项目制切入。

## 第 5-6 周：中国供给侧筛选 / Woche 5-6: Kandidaten- und Partner-Screening

- 建立 20-50 名技工或 3-5 家中国产业合作方的能力档案。
- 收集证书、项目经历、设备经验、工艺视频、安全记录、海外项目经验。
- 用德国客户语言重写能力描述，避免只写“多年经验”。
- 形成第一批 5-10 个可展示样板档案。

## 第 7-8 周：短训模块设计 / Woche 7-8: Kurzqualifizierung

- 设计 2-8 周短训模块：
  - 德国施工/服务流程。
  - 质量记录与验收文档。
  - EHS、PPE、现场安全责任。
  - 客户沟通和基础德语/英语场景句。
  - 工具、材料、标准差异。
- 明确短训只解决能力对齐，不承诺替代正式职业认可。

## 第 9-10 周：合规路径预审 / Woche 9-10: Compliance-Vorprüfung

- 按岗位确认是否为监管职业。
- 评估可能路径：职业经验签证、认可伙伴关系、雇佣前置同意、企业间项目服务等。
- 请移民法/劳动法顾问审阅试点边界。
- 输出每类岗位的红黄绿灯判断。

## 第 11-12 周：试点方案与客户材料 / Woche 11-12: Pilotangebot

- 选择 1-2 个最可行试点场景。
- 准备德国客户版试点方案：岗位、人数、周期、监督方式、交付指标、风险边界。
- 准备中国客户版出海方案：能力要求、资料清单、训练内容、成本结构、时间表。
- 更新网站产业回顾，发布第一期项目观察。

## 成功标准 / Erfolgskriterien

- 至少 3 家德国企业表达继续讨论试点意愿。
- 至少 1 个岗位场景通过初步合规预审。
- 至少 5 个中国技工档案达到德国客户可读标准。
- 形成一个可报价、可访谈、可迭代的试点产品。
